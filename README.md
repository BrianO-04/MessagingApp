# MessagingApp

Direct messaging app written in C using sockets and the TCP protocol

## How to compile 
	$ git clone https://github.com/BrianO-04/MessagingApp.git/
	$ cd MessagingApp	
	$ cmake --build build
## How to run the client 
	$ cd build
	$ ./MessagingApp {Name} {server ip}
## how to run server
	$ cd build
	$ ./MessagingServer
